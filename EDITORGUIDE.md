# Руководство по Editor Mode

## Назначение
`Editor Mode` позволяет редактировать террейн и размещать локальные источники света прямо в текущей сцене.

Редактор использует существующий pipeline движка:
- свет попадает в `sceneLighting.PointLights`
- затем проходит через канонический `lightManager`
- рендерится через обычный `LightingFrame` (без отдельного lighting-пути)

## Быстрый старт
1. Запустите `examples/subnautica`.
2. Нажмите `TAB`, чтобы включить редактор.
3. Выберите режим:
- `T` - террейн
- `L` - свет

## Навигация в редакторе
- `RMB` (удерживать) - осмотр камерой
- `W/A/S/D` - движение по XZ
- `Space` / `Left Ctrl` - вверх / вниз
- `Shift` - ускорение
- `F` - вернуть обзорную editor-камеру

При входе в `Editor Mode` камера автоматически ставится в удобную обзорную позицию.

## Вода в редакторе
- `V` - переключить видимость воды (`WATER ON/OFF`)

Если вода скрыта:
- поверхность воды не рендерится
- подводное затухание и каустика в редакторе отключаются
- это делает террейн/объекты/свет более читаемыми при правке

## Редактирование террейна
### Режимы кисти
- `1` - `Raise` (поднять)
- `2` - `Lower` (опустить)
- `3` - `Smooth` (сгладить)
- `4` - `Flatten` (выровнять к целевой высоте)

### Управление кистью
- `LMB` (удерживать) - непрерывное применение кисти
- `[` / `]` - уменьшить/увеличить радиус кисти
- `-` / `=` - уменьшить/увеличить силу кисти
- `RMB` (удерживать) - временно перейти в режим осмотра без применения кисти

### Поведение кисти
- `Radius` задаёт область влияния вокруг точки попадания курсора в террейн.
- `Strength` задаёт агрессивность изменения высот за кадр.
- Внутри радиуса используется мягкий falloff (влияние сильнее в центре и слабее к краю).
- Непрерывное редактирование при удержании `LMB` сохраняется для всех режимов.

### Smooth
- `Smooth` подтягивает высоты к локальному среднему (3x3), уменьшая резкие пики и ступени.

### Flatten
- В режиме `Flatten` целевая высота берётся из высоты террейна в момент начала удержания `LMB`.
- Пока `LMB` удерживается, target height фиксирован (не «плавает» между кадрами).
- После отпускания `LMB` target сбрасывается и будет заново выбран при следующем нажатии.

### Brush preview
- Превью кисти отображается в мире поверх террейна в точке попадания курсора.
- Цвет превью зависит от режима (`Raise`/`Lower`/`Smooth`/`Flatten`).
- Толщина/выразительность колец усиливается с ростом `Strength`.
- В режиме `Flatten` дополнительно показывается кольцо на целевой высоте, когда target уже зафиксирован.

## Работа со светом
- Переключитесь в режим света: `L`
- Выберите пресет:
- `1` - Малый
- `2` - Средний
- `3` - Большой
- `LMB` по террейну - поставить новый свет
- `LMB` по маркеру света - выбрать свет
- `Ctrl + LMB` по террейну - переместить выбранный свет
- `Delete` - удалить выбранный свет

## Сохранение
- `Ctrl + S` сохраняет:
- террейн: `terrain_map.json`
- lights: `scene_lights.json`

## Важно
- Маркеры света видны только в редакторе.
- Это не отдельная runtime-система света, а editor-визуализация для выбора/перемещения.
- При превышении лимита активных источников часть lights останется в данных сцены, но не активируется в текущем кадре.
